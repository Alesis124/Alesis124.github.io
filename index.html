Sobre Mí

Soy un estudiante de desarrollo de aplicaciones multiplataforma que ha aprendido programación de Java y C#, además de lenguajes de alto nivel web como HTML, CSS y JavaScript.


Soy una persona original y creativa, por eso a la hora de crear un programa o una página web suelo tener algo más de soltura. Otro de mis rasgos importantes es mi lenguaje, ya que hablo como lengua nativa el Español y como lengua secundaria el Inglés, con un poco de Francés.
